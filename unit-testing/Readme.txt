Name Sorter App

This name sorter app created with .Net Core Framework and C# Languages.
To run open Command Prompt, go to folder "win10-x64" and run the command below :
NameSorter.exe "path of unsorted name file".

This App run with 1 parameter, path of directory unsorted name file.
if parameter not supplied it will use default file inside the program.

This application can sort file that have bunch of name inside, it will sort by last name.
Sort result will appear on the screen and will save in new file in "..\\NameSorter\\sorted-names-list.txt".
